Best of 178 candidate topology actions (bus splits), found by simulating each with obs.simulate() and picking the lowest resulting max rho. Applied to the live env → grid back to a safe N-0 state.

3. What this proves

  • Physics engine works: Grid2Op/pandapower power flow runs and responds correctly to topology changes.
  • Redispatch alone is too weak for this scenario (barely moves rho) — topology actions are the real lever. Important for agent tool design.
  • Fast enough for an agent loop: 178 simulations in seconds → can be exposed as a simulate_action tool an LLM calls repeatedly.
  • Built-in baseline: brute-force topology search is the benchmark to beat — agent's edge is narrated reasoning + smarter search on bigger grids where brute force explodes combinatorially.
